# Hadoop HDFS 常用命令

PS：hadoop fs = Hadoop File System Shell

## 1. 查看 Hadoop 版本

```bash
hadoop version
```

## 2. 列出 HDFS 根目录内容

```bash
hadoop fs -ls /
```

## 3. 查看当前挂载的 HDFS 文件系统空间使用情况

```bash
hadoop fs -df hdfs:/
# 生成报告
hadoop dfsadmin -report
```

## 4. 统计匹配路径下的目录数、文件数和字节数

```bash
hadoop fs -count hdfs:/
```

## 5. 运行 HDFS 文件系统检查工具（FSCK）

```bash
hadoop fsck /
```

## 6. 运行集群负载均衡工具

```bash
hadoop balancer
```

## 7. 在 HDFS 的 /user/training 目录下创建名为 hadoop 的目录

（当前用户为 training，因此 /user/training 为你的 HDFS 主目录）

```bash
hadoop fs -mkdir /user/training/hadoop
```

## 8. 将本地 data 目录下的 sample.txt 上传到 HDFS 新建目录

```bash
hadoop fs -put data/sample.txt /user/training/hadoop
```

## 9. 列出该 HDFS 目录内容

```bash
hadoop fs -ls /user/training/hadoop
```

## 10. 将本地整个 retail 目录上传到 HDFS

```bash
hadoop fs -put data/retail /user/training/hadoop
```

## 11. 列出当前用户的 HDFS 主目录

```bash
hadoop fs -ls
```

## 12. 查看 retail 目录占用的 HDFS 空间

```bash
hadoop fs -du -s -h hadoop/retail
```

## 13. 删除 retail 目录中的 customers 文件

```bash
hadoop fs -rm hadoop/retail/customers
```

## 14. 确认文件已被删除

```bash
hadoop fs -ls hadoop/retail/customers
```

## 15. 使用通配符删除 retail 目录下所有文件

```bash
hadoop fs -rm hadoop/retail/*
```

## 16. 清空 HDFS 回收站

```bash
hadoop fs -expunge
```

## 17. 删除整个 retail 目录

```bash
hadoop fs -rm -r hadoop/retail
```

## 18. 再次列出 hadoop 目录

```bash
hadoop fs -ls hadoop
```

## 19. 上传本地 purchases.txt 到 HDFS hadoop 目录

```bash
hadoop fs -copyFromLocal /home/training/purchases.txt hadoop/
```

## 20. 查看 purchases.txt 文件内容

```bash
hadoop fs -cat hadoop/purchases.txt
```

## 21. 将 HDFS 中的 purchases.txt 下载到本地 data 目录

```bash
hadoop fs -copyToLocal hadoop/purchases.txt /home/training/data
```

## 22. 在 HDFS 内部复制 .txt 文件到 hadoop 目录

```bash
hadoop fs -cp /user/training/*.txt /user/training/hadoop
```

## 23. 使用 get 命令下载文件到本地

```bash
hadoop fs -get hadoop/sample.txt /home/training/
```

## 24. 查看 purchases.txt 文件末尾 1KB 内容

```bash
hadoop fs -tail hadoop/purchases.txt
```

## 25. 修改文件权限（默认 666 → 改为 600）

```bash
hadoop fs -ls hadoop/purchases.txt
sudo -u hdfs hadoop fs -chmod 600 hadoop/purchases.txt
```

## 26. 修改文件的所有者和用户组为 root:root

```bash
hadoop fs -ls hadoop/purchases.txt
sudo -u hdfs hadoop fs -chown root:root hadoop/purchases.txt
```

## 27. 仅修改文件所属组为 training

```bash
hadoop fs -ls hadoop/purchases.txt
sudo -u hdfs hadoop fs -chgrp training hadoop/purchases.txt
```

## 28. 重命名/移动目录

```bash
hadoop fs -mv hadoop apache_hadoop
```

## 29. 修改文件副本数为 2

```bash
hadoop fs -setrep -w 2 apache_hadoop/sample.txt
```

## 30. 使用 distcp 在不同 NameNode 之间拷贝数据

```bash
hadoop distcp hdfs://namenodeA/apache_hadoop hdfs://namenodeB/hadoop
```

## 31. 让 NameNode 退出安全模式

```bash
sudo -u hdfs hdfs dfsadmin -safemode leave
```

## 32. 列出所有 HDFS shell 命令

```bash
hadoop fs
```

## 33. 查看帮助文档

```bash
hadoop fs -help
```
